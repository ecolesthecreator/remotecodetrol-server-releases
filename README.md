# RemoteCodetrol Server

Official releases for RemoteCodetrol Server - a macOS menu bar app for iOS development.

## Download

Download the latest version from [Releases](https://github.com/ecolesthecreator/remotecodetrol-server-releases/releases).

## Installation

1. Download the latest `.zip` file
2. Extract and move `RemoteCodetrolServer.app` to your Applications folder
3. Open the app - it will appear in your menu bar

## System Requirements

- macOS 11.0 or later
- Apple Silicon or Intel Mac

## Features

- iOS device pairing
- Automatic server management
- Auto-updates via Sparkle
